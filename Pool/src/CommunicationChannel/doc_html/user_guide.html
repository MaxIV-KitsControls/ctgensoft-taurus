1 - Init

  • Description: This commands re-initialise a device keeping the same network connection.
    After an Init command executed on a device, it is not necessary for client to re-connect to the device.
    This command first calls the device delete_device() method and then execute its init_device() method.
    For C++ device server, all the memory allocated in the nit_device() method must be freed in the delete_device() method.
    The language device desctructor automatically calls the delete_device() method.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    DEV_VOID     : none.
     
  • Command allowed for:
    • Tango::ON
    • Tango::FAULT
    • Tango::UNKNOWN

     


2 - State

  • Description: This command gets the device state (stored in its device_state data member) and returns it to the caller.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    DEV_STATE     : State Code
     
  • Command allowed for:
    • Tango::ON
    • Tango::FAULT
    • Tango::UNKNOWN

     


3 - Status

  • Description: This command gets the device status (stored in its device_status data member) and returns it to the caller.
     
  • Argin:
    DEV_VOID     : none.
     
  • Argout:
    CONST_DEV_STRING     : Status description
     
  • Command allowed for:
    • Tango::ON
    • Tango::FAULT
    • Tango::UNKNOWN

     


4 - Read

  • Description: Atempts to read up to n bytes from the comunication channel. If n is 0(zero), Read() returns an empty array. If n is -1, Read() will return all data available in the channel.
     
  • Argin:
    DEV_LONG     : Number of bytes to read from the comunication channel. 0 will return an empty array. -1 will read all available data
     
  • Argout:
    DEVVAR_CHARARRAY     : The data stream read from the channel
     
  • Command allowed for:
    • Tango::ON

     


5 - Write

  • Description: Sends the given data as parameter to the comunication channel. The result is the number of bytes actually sent to the channel.
     
  • Argin:
    DEVVAR_CHARARRAY     : The data stream to be sent to the comunication channel
     
  • Argout:
    DEV_LONG     : The number of bytes actually sent to the channel
     
  • Command allowed for:
    • Tango::ON

     


6 - WriteRead

  • Description: Sends the given data as parameter to the comunication channel. After this operation completes it performs a read to get all data available in the comunication channel.
     
  • Argin:
    DEVVAR_CHARARRAY     : The data stream to be sent to the comunication channel
     
  • Argout:
    DEVVAR_CHARARRAY     : The data stream read from the channel
     
  • Command allowed for:
    • Tango::ON

     


7 - Open

  • Description: opens the communication channel
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::ON
    • Tango::FAULT
    • Tango::UNKNOWN

     


8 - Close

  • Description: closes the communication channel
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEV_VOID     :
     
  • Command allowed for:
    • Tango::ON
    • Tango::FAULT
    • Tango::UNKNOWN

     


9 - ReadLine

  • Description: Atempts to read characters up to a new line character from the communication channel
     
  • Argin:
    DEV_VOID     :
     
  • Argout:
    DEVVAR_CHARARRAY     : The character read from the communication channel
     
  • Command allowed for:
    • Tango::ON
